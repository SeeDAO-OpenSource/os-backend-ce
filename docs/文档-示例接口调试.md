


## Curl调试接口
1. Linux中调试
```bash
# 使用 GET 方法获取所有 items：
curl http://localhost:3000/items

# 使用 GET 方法获取单个 item（使用实际 ID 替换 <ITEM_ID>）：
curl http://localhost:3000/items/<ITEM_ID>

# 使用 POST 方法创建新的 item：
curl -X POST -H "Content-Type: application/json" -d '{"name": "Sample Item", "description": "Sample item description"}' http://localhost:3000/items

# 使用 PUT 方法更新现有 item（使用实际 ID 替换 <ITEM_ID>）：

curl -X PUT -H "Content-Type: application/json" -d '{"name": "Updated Item", "description": "Updated item description"}' http://localhost:3000/items/<ITEM_ID>

# 使用 DELETE 方法删除现有 item（使用实际 ID 替换 <ITEM_ID>）：

curl -X DELETE http://localhost:3000/items/<ITEM_ID>
```

2. Windows Powershell中调试
2.1 创建
```
$url = "http://localhost:3000/items"
$body = @{
    name = "Sample Item"
    description = "Sample item description" 
} | ConvertTo-Json

$headers = @{
    "Content-Type" = "application/json" 
}

Invoke-WebRequest -Uri $url -Method Post -Headers $headers -Body $body

```
